Santa Stacker is a festive, single-tap arcade game built entirely in a single HTML file. Help Santa stack gifts as high as possible against the skylines of famous cities, unlock sky secrets, and share holiday greetings with friends!

Objective: Stack the moving gift blocks on top of each other.

Controls: Press SPACEBAR or TAP the screen to drop a gift.

The Catch: Any part of the gift that overhangs the stack gets sliced off! The higher you go, the faster it gets.

Goal: Deliver expensive gifts, reach the stars, and unlock visual Easter eggs.
